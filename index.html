The three pillars of Network Guardian scoring.

Scoring logic is covered by tests (Safe / Caution / Unsafe thresholds) and backed by Postgres tables: wifi_reports, network_reputation, and telemetry_metrics.

1. Security

Spots spoofed SSIDs, malicious gateways, and obvious MITM / DNS hijack patterns before a user trusts the network.

  • SSID/BSSID integrity checks
  • Route path / ASN sanity
  • Heuristics for captive portals & injection

2. Stability

Evaluates success ratio, latency, jitter, and loss to tell users whether the network can handle real work.

  • Telemetry event ingestion via /report
  • Percentiles served via /stats/stability
  • Future: region & plan-aware stability profiles

3. Community

Aggregates crowdsourced trust over time into a social-graph-style reputation for every BSSID worldwide.

  • Global BSSID reputation graph
  • Community weighted scores (EMAs post-MVP)
  • Operator & location-aware patterns

FastAPI backend: clean, opinionated, and ready for production.

Deployed as a containerized FastAPI service behind a load balancer. Uses PostgreSQL + SQLAlchemy 2.x and a minimal, test-driven scoring core.

Core endpoints

All responses are JSON. OpenAPI docs are exposed at /docs.

Method Endpoint Description
GET /health Liveness probe used by orchestration and uptime checks.
GET /config Returns community weights, scoring flags, and feature switches.
GET /public-ip Tags reports with public IP / ASN / operator name for reputation modeling.
POST /report Ingests a Wi-Fi scan payload, persists it to wifi_reports, recomputes network_reputation, and returns the updated score.
GET /report/{id} Fetches a single stored report for debugging and observability.
GET /network/{bssid} Returns the current reputation and metadata for a single BSSID.
GET /networks Provides a catalog of known networks with aggregated Trust Scores.
GET /stats/stability Exposes latency / jitter / loss percentiles for stability analytics.

Tests cover the scoring logic, /report persistence, and /network / /networks lookups ‚Äî so you can ship with confidence.
